1. Clone the repository locally (from git client).
2. Touch <filename> to add a file to the repository (add whatever you want).
3. git add to add the file to your repo.	
4. git status to see status of file changes.
5. Try a few cycles of:
 - git status
 - edit <filename>
 - git add <filename>
 - git status
 - git commit -m "message"
 - git push
 - git status


Branch and checkout
1. git branch <branchname>
2. git checkout <branchname> 
this will switch you to <branchname>
3. edit a file and git add it
4. git commit -m
5. git push --set-upstream origin firstbranch
